## How to create a new story

1. Copy [STORY_TEMPLATE.md](./STORY_TEMPLATE.md) and rename (e.g. `S1-2-implement-storage-service.md`).
2. Fill in ID, title, sprint, and acceptance criteria (match sprint table).
3. Break down into tasks and set status to **Draft** or **In progress**.
